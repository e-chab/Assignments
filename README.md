# Trick or Treat - Performance Analysis

This project implements and compares different versions of the "Trick or Treat" algorithm, which finds the best sequence of candies.

## Project Structure

- **[in_c/](in_c/)** - Sequential C implementation
  - Simple reference implementation using C standard library no real test made

- **[in_cpp/](in_cpp/)** - Sequential C++ implementation 

- **[openMP/](openMP/)** - Parallel OpenMP implementation
  - Multi-threaded version of a brute force algorithm using OpenMP pragmas

- **[input_onpenMP/](input_onpenMP/)** - Input reading parallelization experiments
