# Implementation C++ - Sequential

This directory contains a sequential implementation of the trick-or-treat problem in C++ using sliding window algorithm.

## Files

- **`trick_or_treat.cpp/h`** : Sequential implementation with O(n) sliding window algorithm
- **`test.cpp`** : Test program
- **`test_benchmark.cpp`** : Google Benchmark for performance analysis

## Usage

### Simple test
```bash
make test
./test [input_file]
```

### Benchmark
```bash
make benchmark
./benchmark
```

Default input: `../input_10000_pirecas.txt`
